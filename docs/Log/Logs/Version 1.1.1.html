Version 1.1.1¶

Online/Offline¶

Switching between online/offline changes as follows¶

Version 1.1.0 (before)

>>> from PyJEM import detecotor
>>> detector.offline()

Version 1.1.1 (current)

>>> from PyJEM.offline import detector

This change removes the following features.

  • offline()

  • online()

  • isonline()

  • get_connect_info()
