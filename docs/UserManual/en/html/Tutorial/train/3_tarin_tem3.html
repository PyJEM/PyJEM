Sample code (TEM3 package)¶

  • Prerequisites

    • TEMCenter is running (in the case of online)

    • TEMServer is running (in the case of online)

    • Ver.1.1.0 (Operation check environment)

1. Get current mag value¶

  • Use GetMagValue() function to get the value.

import TEM3¶

[1]:

# online
from PyJEM import TEM3
TEM3.connect()

## offline
#from PyJEM,offline import TEM3
#TEM3.connect()

Create an instance of the class to use¶

[2]:

eos = TEM3.EOS3()

call get function¶

[3]:

eos.GetMagValue()

[3]:

[200, 'X', 'X200']

2. Change the magnification¶

  • The magnification can be changed with SetSelector() defined in TEM3.EOS3().

    • SetSelector () has one argument and will be changed to a magnification according to the entered value.

    • The input value is 0 to 40 in index.

  • The Set function has no return value. You can check the changed magnification with GetMagValue ().

call change mag function¶

  • change to 1 (index) mag value.

[19]:

eos.SetSelector(1)

Check TEM state¶

[6]:

eos.GetMagValue()

[6]:

[250, 'X', 'X250']

3. Increase the magnification by 10 steps from the current magnification using the for statement¶

  • Use “for” statement

    >>> for i in range(10):

  • Use function UpSelector() to increase the magnification one step at a time.

Execute¶

[20]:

for i in range(10):
    eos.UpSelector()
    print(eos.GetMagValue())

4. Define 3. as a function and execute it.¶

  • Define the number of loops as an argument

  • Define the function name as main.

define function¶

[23]:

def main(loop):
    for i in range(loop):
        eos.UpSelector()
        print(eos.GetMagValue())

execute function¶

[24]:

# loop
main(10)
