A1. 利用可能な検出器名の取得¶

  • 検出器の制御はPyJEM.detectorパッケージを利用します。

  • 利用可能な検出器の取得関数は、get_attached_detector()です。

パッケージの読み込み¶

[1]:

from PyJEM.offline import detector   #offline
#from PyJEM import detector           #online

0 : camera
1 : STEM Image

利用可能な検出器名を取得¶

[2]:

detector.get_attached_detector()

[2]:

['camera', 'STEM Image']

A2. cameraの検出器情報の取得¶

  • 検出器を指定する方法は、detector.Detector()の引数に検出器を入力することでその検出器のインスタンスが作成されます。

  • 検出器名は、大文字・小文字を正しく入力してください。

  • 検出器情報は、get_detectorsetting()で取得できます。

パッケージの読み込み¶

[3]:

from PyJEM.offline import detector   #offline
#from PyJEM import detector           #online

検出器を指定¶

[4]:

cam = detector.Detector("camera")

camera is Correct!

cam.get_detectorsetting()

A3. Gainを変更する¶

  • cameraのGain値を変更する。

  • Gainの変更は、set_gainindex()

  • 引数に、gainのindex値を入力する

パッケージの読み込み¶

[7]:

from PyJEM.offline import detector   #offline
#from PyJEM import detector           #online

検出器を指定¶

[8]:

cam = detector.Detector("camera")

Gain値を変更¶

[5]:

cam.set_gainindex(1234)

[5]:

{'GainIndex': 1234}

Q4. 指定した検出器で画像の撮影・取得¶

  • cameraに対して行う

  • snapshot()で画像の撮影を行う。

  • 引数には、撮影した画像を保存する拡張子

パッケージの読み込み¶

[7]:

from PyJEM.offline import detector   #offline
#from PyJEM import detector           #online

検出器を指定¶

[6]:

cam = detector.Detector("camera")

camera is Correct!

画像の撮影¶

[8]:

%matplotlib inline
cam.snapshot("jpg",show=True)
