Activate python environment.¶

conda activate¶

Use python3.5¶

  1. Open the “Anaconda Prompt”

  2. Write “conda activate vjem35” on the command line.

Use python3.7¶

  1. Open the “Anaconda Prompt”

  2. Write “conda activate vjem37” on the command line.

Use python3.8¶

  1. Open the “Anaconda Prompt”

  2. Write “conda activate vjem38” on the command line.

Example¶

Activate python3.5 environment.

(base) C:\Users\JEOL\Desktop>conda activate vjem35
(vjem35) C:\Users\JEOL\Desktop>python
Python 3.5.3 | packaged by conda-forge | (default, May 12 2017, 16:16:49) [MSC v.1900 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>

Online¶

TEM3 package¶

Load the TEM3 package to control TEM.

1. import¶

The below code will load the TEM3 package.

>>> from PyJEM import TEM3

2. connect to TEM¶

Execute the following code to connect to TEM.
There are 2 ways to connect to the TEM.

Execute the connect function

>>> TEM3.connect()
True
Ready Communication
Create class instance
ex) create Apt3 instance
>>> TEM3.Apt3()
True
Ready Communication

3. Check communication status¶

You can check the communication status with TEM with the following code.
The return value is Boolean. True is output if communication is successful.
>>> TEM3.isconnect()
True

If communication is possible, the function can be executed from the interface documents.

detector package¶

Load the detector package to control camera/detector.
With this package, you can use JEOL’s camera, stem detector, and its associated features.

1. import¶

The below code will load the detector package.

>>> from PyJEM import detector

2. Check communication status¶

You can check the communication status with TEM with the following code.

>>> detector.get_connect_info()
("detector", "online", "127.0.0.1")

From the above execution result, “127.0.0.1” displays IP address of the connection destination PC.

  • If “offline” is displayed, change it to “online” from the code below.

>>> detector.online()
("detector", "online", "127.0.0.1")

  • If IP address is different, please change it from the code below.

>>> detector.change_ip(...)
("detector", "online", "...")
