More customize¶

Online / offline¶

ver. 1.1.0.*¶

  • online

>>> from PyJEM import detector
>>> detector.online()

  • offline

>>> from PyJEM import detector
>>> detector.offline()

Other version¶

  • online

>>> from PyJEM import detector

  • offline

>>> from PyJEM.offline import detector

Change IP address¶

Change ipaddress when PyJEM is configured outside of TEMPC.

TEM3¶

There is no way to change it.

Other package¶

Take the detector package as an example.

>>> from PyJEM import detector
>>> detector.set_ip("127.0.0.1")
